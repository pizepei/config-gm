# config-gm
通用配置集合（方便移植）
###
增加了自动合并配置方法
###
配置方式重新规划
* MICROSERVICE 配置 系统级别
* 按照包模块显示配置和保存配置（系统下可能存在多个服务模块中有相同的包模块配置）
* 准备一个文件 部署时触发
* 部署流程
    1、git clone   
    2、composer install  
    3、部署项目进入到对应目录vendor搜索包的configTpl.php文件 
    4、一一返回个前端 
    5、部署人员安置配置模板填写，点击保存文件内容就会保存到数据库（当系统部署成功后第一次运行时会根据部署配置请求配置中心获取合并的完整的config内容生成config.php文件）
